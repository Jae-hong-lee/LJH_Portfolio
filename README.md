# 재홍 포트폴리오 사이트

## 📌 나만의 포트폴리오 사이트

- 반응형 웹 사이트 기획부터 배포
- `GitHub Action` 과 `AWS S3` 을 연동하여 CI/CD 파이프라인 구축
- `Notion API` 를 통해 Notion DB API 호출 `fetch`
- DarkMode 구현
- Section 이동 구현

## ✔️ Commit Message Convention

| Tag Name | Description                           |
| -------- | ------------------------------------- |
| feat     | 새로운 기능을 추가할 경우             |
| fix      | 버그를 고친 경우 or 수정사항 있을경우 |
| design   | CSS 등 사용자 UI 디자인 변경          |
| comment  | comment 추가                          |

`Ex ) feat:로그인, 회원가입 UI `
